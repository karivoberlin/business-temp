# KARIVO 2.0 – Phase 1

Neue deutsche Premium-Startseite mit:
- komplett neuem Hero
- schwarzer Glaskugel mit Orbit-Linien
- Maus-Parallax und Scroll-Reaktion
- cleanem Premium-Look
- allgemeinen Webdesign-Texten ohne Spezialisierung
- Pakete, Prozess und Kontaktformular

## Starten
```bash
npm install
npm run dev
```

## Deployment
GitHub hochladen → Vercel verbinden → deployen.
